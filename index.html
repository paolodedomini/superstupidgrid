Una piccola libreria SuperStupida per creare pagine web in modo Superveloce.
Perchè non sempre abbiamo bisogno di Bootstrap.
Perchè non sempre abbiamo bisogno di Tailwind.

Se sai scrivere css e vuoi qualcosa che velocizzi e imposti il lavoro di sviluppo della pagina web, questo approccio potrebbe esserti d'aiuto. Oppure no. Per me lo è.

Tutto piuttosto semplice (e un pò stupido). Dobbiamo creare una griglia per gestire i contenuti? Creiamo una classe e includiamo il mixin container. Se lo si include senza parametri si crea di default un container-fluid. Una volta creata la classe container, al suo interno creiamo delle classi colonna. Due le possibilità: creare in automatico una classe tramite il mixin col oppure utilizzare la classe col-custom. Il primo mixin accetta un numero come parametro che corrisponde alla quantità di colonne per riga. Il secondo mixin accetta un nome per creare il tipo di colonne richiesto, in questo caso è necessario impostare una dimensione.
